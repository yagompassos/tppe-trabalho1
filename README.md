# Árvore B - TP1

## Como rodar

Você precisa ter [python 3](https://www.python.org/downloads/) instalado, e seu gerenciador de pacotes pip.

1. Baixe as dependèncias:

```bash
pip install icontract
```

2. Rode o arquivo principal:

```bash
python main.py
```
